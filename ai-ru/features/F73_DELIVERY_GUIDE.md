# F73: Delivery Guide

**Version:** 0.68.0
**Date:** 2026-03-20

## Summary

Embedded bilingual (RU/EN) delivery process guide at `/guide` page. Documents the full Kanban/Flow pipeline from Idea to Done with role cards, quality controls, and links to Lead Board screens.

## Key Features

- 8-stage pipeline visualization (interactive, clickable)
- Cross-cutting quality control layer
- 5 role reference cards (PO, DM, TL, Pipeline Executors, DevOps)
- Rules & constraints section (estimation, hierarchy, grades)
- Bilingual RU/EN with language switcher
- Sidebar navigation with scroll tracking
- Deep links via URL hash fragments
