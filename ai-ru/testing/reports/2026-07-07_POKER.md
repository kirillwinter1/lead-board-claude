## Root Cause главного бага

Миграция **V45** (`drop_legacy_public_tables`, коммит c09f3c3) удалила таблицы `poker_sessions/poker_stories/poker_votes` из схемы `public` — они существуют только в tenant-схемах. REST-запросы получают tenant через `TenantFilter` (subdomain / X-Tenant-Slug), но **`PokerWebSocketHandler` работает в потоках Tomcat WebSocket без `TenantContext`** (в пакете poker нет ни одного упоминания TenantContext). Hibernate ставит `search_path=public` → каждая WS-операция падает:

```
org.springframework.dao.InvalidDataAccessResourceUsageException:
ERROR: relation "poker_sessions" does not exist
  at com.leadboard.poker.websocket.PokerWebSocketHandler.handleJoin(PokerWebSocketHandler.java:145)
```

Дополнительно: браузерный WebSocket API не позволяет передать заголовок `X-Tenant-Slug` при handshake — даже починка потоков потребует другого механизма резолва tenant (по subdomain из Origin/Host или по tenant_id сессии пользователя).

Следствие: фича сломана **во всех окружениях** (локально и на проде) с момента V45, а не только «отложена из-за багов Jira».

## Recommendations

1. **P0:** Резолвить tenant для WS: сохранять tenant при handshake (HandshakeInterceptor кладёт tenantId в attributes сессии) и оборачивать каждую операцию в `TenantContext.setTenant(...)/clear()`. Либо перевести poker на REST-polling до починки.
2. **P0:** Определять участника и facilitator на сервере: accountId — из session cookie, isFacilitator — сравнением с `session.facilitatorAccountId`. Убрать доверие к клиентскому payload; добавить проверки на REST-мутации + стейт-гарды (final только из REVEALED).
3. **P1:** PokerJiraService → JiraConfigResolver (+ выбор проекта при multi-project, F64).
4. **P1:** ELIGIBLE_STATUSES → категории статусов из WorkflowConfigService.
5. **P2:** Синхронизировать спеку F23 с кодом, добавить удаление story/сессии, exponential backoff + лимит reconnect, `ч` → `ч`, предзаполнение финальной оценки медианой.
6. Учитывая объём (real-time ядро сломано + security), рассмотреть редизайн в рамках BF20 «Planning Poker v2» вместо точечных фиксов.
